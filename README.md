# Studypoint

StudyPoint – Smart Study Planner Website
Targeted Users:
School & college students
Competitive exam aspirants
Students who want to manage study time, tasks, and attendance effectively

Team Members & Roles:

Rushikesh Salvi – Frontend Developer & UI Designer

Shailesh – Backend Developer

Project Description:
StudyPoint is a smart study planner website designed to help students manage their academic life
efficiently. It provides features like task management, subject organization, timetable scheduling,
and attendance tracking.
The goal is to improve productivity and consistency.

Requirements & AI Tools Used
Functional Requirements:
- Dashboard
- Task Manager
- Subject Manager
- Timetable Scheduler
- Attendance Tracker
- Responsive Design
Non-Functional Requirements:
- Easy UI
- Fast Loading
- Clean Design
- Cross-browser compatibility
